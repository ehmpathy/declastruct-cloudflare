# research: remote access

research on remote access required to fulfill the wish, vision, and criteria.

---

## summary

| repository | purpose | access method | automatable? |
|------------|---------|---------------|--------------|
| cloudflare zones api | zone CRUD | REST via SDK | full |
| cloudflare dns api | dns record CRUD | REST via SDK | full |
| cloudflare registrar api | domain management | REST via SDK | partial (config only) |
| cloudflare intel whois api | domain state detection | REST via SDK | full |

**key insight**: registration and transfer require dashboard. whois api enables smart detection for appropriate guidance.

## api.1: zones

### contract

| operation | method | endpoint |
|-----------|--------|----------|
| list | GET | /zones |
| create | POST | /zones |
| get | GET | /zones/{zone_id} |
| edit | PATCH | /zones/{zone_id} |
| delete | DELETE | /zones/{zone_id} |

### sdk usage

```ts
// list
const zones = await client.zones.list();

// create
await client.zones.create({
  account: { id: accountId },
  name: 'example.com',
  type: 'full',
});

// get
const zone = await client.zones.get(zoneId);

// edit
await client.zones.edit(zoneId, { paused: true });

// delete
await client.zones.delete(zoneId);
```

## api.2: dns records

### contract

| operation | method | endpoint |
|-----------|--------|----------|
| list | GET | /zones/{zone_id}/dns_records |
| create | POST | /zones/{zone_id}/dns_records |
| get | GET | /zones/{zone_id}/dns_records/{record_id} |
| update | PUT | /zones/{zone_id}/dns_records/{record_id} |
| patch | PATCH | /zones/{zone_id}/dns_records/{record_id} |
| delete | DELETE | /zones/{zone_id}/dns_records/{record_id} |

### sdk usage

```ts
// list
const records = await client.dns.records.list({ zone_id: zoneId });

// create
await client.dns.records.create({
  zone_id: zoneId,
  type: 'A',
  name: 'api',
  content: '192.168.1.1',
  ttl: 300,
  proxied: true,
});

// update
await client.dns.records.update(recordId, {
  zone_id: zoneId,
  type: 'A',
  name: 'api',
  content: '192.168.1.2',
});

// delete
await client.dns.records.delete(recordId, { zone_id: zoneId });
```

### constraints

- A/AAAA + CNAME conflict rejected by api
- TTL minimum 60s (30s for enterprise)
- proxied only valid for A, AAAA, CNAME

## api.3: registrar

### contract

| operation | method | endpoint | automatable? |
|-----------|--------|----------|--------------|
| list | GET | /accounts/{account_id}/registrar/domains | yes |
| update | PUT | /accounts/{account_id}/registrar/domains/{domain} | yes |
| register | n/a | dashboard only | no |
| transfer | n/a | dashboard only | no |

### sdk usage

```ts
// list
const domains = await client.registrar.domains.list({ account_id: accountId });

// update config
await client.registrar.domains.update(domainName, {
  account_id: accountId,
  auto_renew: true,
  locked: true,
  privacy: true,
});
```

### updatable options

| option | type | description |
|--------|------|-------------|
| auto_renew | boolean | auto-renew before expiration |
| locked | boolean | registrar lock (prevent transfer) |
| privacy | boolean | whois privacy protection |

## api.4: intel whois

### contract

| operation | method | endpoint |
|-----------|--------|----------|
| get | GET | /accounts/{account_id}/intel/whois?domain={domain} |

### sdk usage

```ts
const whois = await client.intel.whois.get({
  account_id: accountId,
  domain: 'example.com',
});
```

### response shape

| field | type | description |
|-------|------|-------------|
| found | boolean | whether domain is registered |
| registrar | string | current registrar name |
| nameservers | string[] | current nameservers |
| registrant | object | registrant info (if available) |

### use case for guidance

```ts
// in setDomainRegistration
const whois = await client.intel.whois.get({ account_id: accountId, domain });

if (whois.found) {
  // domain registered elsewhere → throw transfer guidance
  BadRequestError.throw(`transfer required`, { registrar: whois.registrar });
} else {
  // domain unregistered → throw purchase guidance
  BadRequestError.throw(`purchase required`, { domain });
}
```

## automation limits

| capability | automatable? | fallback |
|------------|--------------|----------|
| zone CRUD | yes | n/a |
| dns record CRUD | yes | n/a |
| registration config | yes | n/a |
| domain registration | no | HelpfulError purchase guidance |
| domain transfer | no | HelpfulError transfer guidance |

**this validates the vision**: HelpfulError guidance pattern required for registration/transfer.
