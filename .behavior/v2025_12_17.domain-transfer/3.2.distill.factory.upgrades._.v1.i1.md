### credentials blocker detail

integration and acceptance tests require cloudflare API credentials.

**resolution** (human action required):
```sh
# 1. init ehmpath keyrack (one-time per machine)
npx rhachet roles init --repo ehmpathy --role mechanic --init keyrack.ehmpath

# 2. unlock keyrack
npx rhx keyrack unlock --owner ehmpath --prikey ~/.ssh/ehmpath --env all

# 3. fill credentials
./.agent/repo=.this/role=any/skills/keyrack.fill.sh --env test
```

**impact** (before credentials):
- unit tests pass (105)
- integration tests fail
- acceptance tests fail

**impact** (after credentials):
- all tests runnable
- full verification enabled
