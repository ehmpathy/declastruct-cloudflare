# distill: experience reproductions

distillation of user experience reproductions for domain-transfer behavior.

---

## experience reproductions

| experience | entry point | user actions | expected outcome | test type |
|------------|-------------|--------------|------------------|-----------|
| declare zone | resources.ts | new DeclaredCloudflareDomainZone({...}) | zone instance created | unit |
| declare dns record | resources.ts | new DeclaredCloudflareDomainDnsRecord({...}) | record instance with zone ref | unit |

## reproduction feasibility

### experience: declare zone

**test utilities**: jest, domain-objects instantiation
**setup**: none
**sketch**:
```ts
it('should be instantiable', () => {
  const zone = new DeclaredCloudflareDomainZone({
    name: 'example.com',
    type: 'full',
  });
  expect(zone.name).toEqual('example.com');
});
```
**feasible**: yes

### experience: declastruct plan

**test utilities**: execSync, fs, jest acceptance config
**setup**: build project, sample resources file
**sketch**:
```ts
const planFile = join(testDir, 'plan.json');
execSync(`npx declastruct plan --wish ${resourcesFile} --into ${planFile}`);
expect(existsSync(planFile)).toBe(true);
```
**feasible**: yes

### experience: transfer guidance

**test utilities**: getMockedCloudflareApiContext, getError
**setup**: mock intel.whois.get response
**sketch**:
```ts
context.cloudflare.client.intel = {
  whois: { get: jest.fn().mockResolvedValue({ found: true, registrar: 'GoDaddy' }) },
} as any;
const error = await getError(setDomainRegistration({ findsert: reg }, context));
expect(error).toBeInstanceOf(BadRequestError);
expect(error.message).toContain('transfer required');
expect(error.message).toContain('GoDaddy');
```
**feasible**: yes

### experience: purchase guidance

**test utilities**: getMockedCloudflareApiContext, getError
**setup**: mock intel.whois.get response
**sketch**:
```ts
context.cloudflare.client.intel = {
  whois: { get: jest.fn().mockResolvedValue({ found: false }) },
} as any;
const error = await getError(setDomainRegistration({ findsert: reg }, context));
expect(error).toBeInstanceOf(BadRequestError);
expect(error.message).toContain('purchase required');
```
**feasible**: yes

## gaps

### gap detail: credentials

**what is blocked**: integration and acceptance tests that hit live cloudflare API

**what is required**: human must provision cloudflare credentials:
```sh
npx rhx keyrack unlock --owner ehmpath --prikey ~/.ssh/ehmpath --env all
./.agent/repo=.this/role=any/skills/keyrack.fill.sh --env test
```

**is this a blocker**: yes for full verification, but:
- unit tests can verify logic via mocks
- acceptance tests have conditional skip when credentials absent
