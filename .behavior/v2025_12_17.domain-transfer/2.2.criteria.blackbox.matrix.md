# coverage matrix: declastruct-cloudflare domain management

matrix = independent variables × dependent variables

---

## dimensions (independent variables)

### dim.1 = resource type

| value | description |
|-------|-------------|
| zone | cloudflare dns zone (container for records) |
| dns_record | individual dns record within a zone |
| registration | domain registrar-level ownership record |

### dim.2 = operation

| value | description |
|-------|-------------|
| getAll | list all resources |
| set.findsert | find or create (idempotent insert) |
| set.upsert | update or create (overwrite if exists) |
| del | remove resource |

### dim.3 = resource state

| value | description |
|-------|-------------|
| exists | resource already present in cloudflare |
| absent | resource not present in cloudflare |

### dim.4 = credentials state

| value | description |
|-------|-------------|
| valid | api token works |
| invalid | api token fails authentication |
| expired | api token was valid, now fails |

### dim.5 = domain location (registration only)

| value | description |
|-------|-------------|
| in_cloudflare | domain registered with cloudflare registrar |
| elsewhere | domain registered with another registrar (e.g., godaddy) |
| unregistered | domain not registered anywhere |

---

## outcomes (dependent variables)

| outcome | description |
|---------|-------------|
| success | operation completes, resource returned |
| created | resource did not exist, now exists |
| updated | resource existed, values changed |
| deleted | resource existed, now removed |
| noop | operation completes, no change (idempotent) |
| empty_list | getAll returns [] |
| error.auth | authentication failed |
| error.badrequest | invalid input or constraint violated |
| guidance.transfer | BadRequestError with transfer steps |
| guidance.purchase | BadRequestError with purchase steps |
| rejected | cloudflare rejects due to constraint (e.g., A+CNAME conflict) |

## summary

**coverage**: 32 explicit combinations across 7 usecases

**gaps accepted**: 4 minor gaps with clear disposition

**critical paths covered**:
- all CRUD operations for zones and dns records
- registration management for domains in cloudflare
- guided workflows for transfer and purchase
- error handle for auth, validation, and api failures
- idempotency for all mutation operations
- drift detection for manual dashboard changes

**the matrix confirms**: criteria cover the full surface area of the vision without redundancy.
