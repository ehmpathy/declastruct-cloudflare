# distill: domain

distillation of declastruct domain.objects and domain.operations for domain-transfer wish.

---

## usecases

### usecase.1: zone management

```ts
// declare zone
const zone = new DeclaredCloudflareDomainZone({
  name: 'mhember.com',
  type: 'full',
});

// findsert (find or create)
const result = await setDomainZone({ findsert: zone }, context);

// upsert (update or create)
const result = await setDomainZone({ upsert: zone }, context);

// delete
await delDomainZone({ by: { unique: { name: 'mhember.com' } } }, context);
```

### usecase.2: dns record management

```ts
// declare record
const record = new DeclaredCloudflareDomainDnsRecord({
  zone: refByUnique(zone),
  name: 'api.mhember.com',
  type: 'A',
  content: '192.168.1.100',
  ttl: 300,
  proxied: true,
});

// findsert
const result = await setDomainDnsRecord({ findsert: record }, context);

// delete
await delDomainDnsRecord({ by: { unique: { zone: { name: 'mhember.com' }, name: 'api.mhember.com', type: 'A', content: '192.168.1.100' } } }, context);
```

### usecase.3: registration management

```ts
// declare registration
const reg = new DeclaredCloudflareDomainRegistration({
  name: 'mhember.com',
  autoRenew: true,
  locked: true,
  privacyProtection: true,
});

// findsert (may throw guidance)
const result = await setDomainRegistration({ findsert: reg }, context);
// if domain in cloudflare: returns registration
// if domain elsewhere: throws transfer guidance
// if domain unregistered: throws purchase guidance
```

## domain objects

### DeclaredCloudflareDomainZone

```ts
interface DeclaredCloudflareDomainZone {
  id?: string;  // readonly, cloudflare-assigned
  name: string;  // immutable, unique key
  type: 'full' | 'partial';
}

class DeclaredCloudflareDomainZone
  extends DomainEntity<DeclaredCloudflareDomainZone>
{
  public static primary = ['id'] as const;
  public static unique = ['name'] as const;
  public static readonly = ['id'] as const;
}
```

### DeclaredCloudflareDomainDnsRecord

```ts
interface DeclaredCloudflareDomainDnsRecord {
  id?: string;  // readonly, cloudflare-assigned
  zone: RefByUnique<typeof DeclaredCloudflareDomainZone>;
  name: string;  // full record name
  type: DeclaredCloudflareDomainDnsRecordType;
  content: string;
  ttl: number;  // 1 = auto, or 60-86400
  proxied?: boolean;  // A, AAAA, CNAME only
  priority?: number;  // MX, SRV only
}

class DeclaredCloudflareDomainDnsRecord
  extends DomainEntity<DeclaredCloudflareDomainDnsRecord>
{
  public static primary = ['id'] as const;
  public static unique = ['zone.name', 'name', 'type', 'content'] as const;
  public static readonly = ['id'] as const;
}
```

### DeclaredCloudflareDomainRegistration

```ts
interface DeclaredCloudflareDomainRegistration {
  id?: string;  // readonly, cloudflare-assigned
  name: string;  // domain name, immutable
  autoRenew: boolean;
  locked: boolean;
  privacyProtection: boolean;
}

class DeclaredCloudflareDomainRegistration
  extends DomainEntity<DeclaredCloudflareDomainRegistration>
{
  public static primary = ['id'] as const;
  public static unique = ['name'] as const;
  public static readonly = ['id'] as const;
}
```

```ts
const DeclaredCloudflareDomainZoneDao = genDeclastructDao<
  typeof DeclaredCloudflareDomainZone,
  ContextCloudflareApi & ContextLogTrail
>({
  dobj: DeclaredCloudflareDomainZone,
  get: {
    one: {
      byPrimary: (input, context) => getOneDomainZone({ by: { primary: input } }, context),
```

## relationships

### treestruct (hierarchy)

```
Zone
└─ DnsRecord (many, via zone ref)
   ├─ A records
   ├─ AAAA records
   ├─ CNAME records
   ├─ MX records
   └─ ...

Registration
└─ related to Zone (same domain name)
```

### dependencies

| from | to | type |
|------|-----|------|
| DnsRecord | Zone | required (belongs to) |
| Registration | Zone | soft (same domain) |
| WhoisRecord | none | independent lookup |

### apply order

1. zones first (container)
2. dns records second (depend on zones)
3. registrations last (may throw guidance)

## composition

```
declastruct plan/apply

resources.ts
├─ getProviders() → [cloudflareProvider]
└─ getResources() → [zone, ...dnsRecords, registration]

declastruct core
├─ resolves references (zone → dns records)
├─ orders operations (zones before records)
├─ computes diff (declared vs actual)
└─ applies changes via dao interface

cloudflareProvider
├─ DeclaredCloudflareDomainZoneDao
├─ DeclaredCloudflareDomainDnsRecordDao
└─ DeclaredCloudflareDomainRegistrationDao
```

## guidance flow

```
setDomainRegistration({ findsert: registration })
│
├─ getOneDomainRegistration({ by: { unique: { name } } })
│  │
│  ├─ found? → return (findsert) or update (upsert)
│  │
│  └─ not found?
│     │
│     └─ getOneDomainWhoisRecord({ domain: name })
│        │
│        ├─ whois.found? → BadRequestError: transfer guidance
│        │
│        └─ !whois.found? → BadRequestError: purchase guidance
```
