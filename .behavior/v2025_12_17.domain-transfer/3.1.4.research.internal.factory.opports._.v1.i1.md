# research: factory opportunities

research on what improvements would help go faster for domain-transfer implementation.

---

## the constraint

### the bottleneck

**credential provisioning** — can't verify integration/acceptance tests until human provisions cloudflare credentials via keyrack.

### impact

| without credentials | with credentials |
|---------------------|------------------|
| unit tests pass (105) | unit tests pass |
| integration tests fail | integration tests run |
| acceptance tests fail | acceptance tests run |
| can verify shape/logic | can verify api behavior |

### if fixed

if credentials were provisioned, the factory would be fully operational. no other blockers exist.
