### 4. idempotency holds (rule.require.idempotent-procedures)

```ts
  // if zone found
  if (zoneFound) {
    // findsert: check for attribute diff, then return extant
    if (input.findsert) {
      const hasTypeDiff = zoneDesired.type !== zoneFound.type;
      const hasPausedDiff =
        zoneDesired.paused !== undefined &&
        zoneDesired.paused !== zoneFound.paused;

      if (hasTypeDiff || hasPausedDiff)
        BadRequestError.throw(
          'cannot findsert zone; zone exists with different attributes',
          { zoneDesired, zoneFound, diffs: { type: hasTypeDiff, paused: hasPausedDiff } },
        );

      return zoneFound;
    }
```

**why it holds**: findsert checks for diff, returns extant if no diff, throws if diff. no duplicate creation. idempotent.
