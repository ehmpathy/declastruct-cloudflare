# research.domain: declastruct-cloudflare domain management

## domain objects

### entity: Zone

**cloudflare api shape** [1]:
```typescript
{
  id: string;                    // unique identifier
  name: string;                  // domain name (e.g., "example.com")
  type: 'full' | 'partial' | 'secondary';
  account: { id: string; name: string };
  status: 'active' | 'pending' | 'moved' | 'deleted';
  name_servers: string[];        // assigned cloudflare nameservers
  activated_on: string | null;   // timestamp
  created_on: string;            // timestamp
  modified_on: string;           // timestamp
  development_mode: number;
  meta: object;
  plan: object;
}
```

**unique key**: `name` (domain name is globally unique per account)
**primary key**: `id`

---

### entity: DnsRecord

**cloudflare api shape** [2]:
```typescript
{
  id: string;                    // unique identifier
  zone_id: string;               // parent zone
  name: string;                  // record name (e.g., "api.example.com")
  type: 'A' | 'AAAA' | 'CNAME' | 'MX' | 'TXT' | 'NS' | 'SRV' | ...;
  content: string;               // record value
  ttl: number;                   // 1 = automatic, 60-86400
  proxied: boolean;              // cloudflare proxy enabled
  proxiable: boolean;            // can be proxied
  priority?: number;             // for MX, SRV records
  comment?: string;              // administrative note
  tags?: string[];               // labels
  created_on: string;            // timestamp
  modified_on: string;           // timestamp
}
```

**unique key**: composite of `(zone_id, name, type)` — same name can have multiple types [2]
**primary key**: `id`

**record type constraints** [2]:
- A/AAAA cannot coexist with CNAME on same name
- NS records cannot share name with other types

### entity: Registration (Domain)

**cloudflare api shape** [3]:
```typescript
{
  id: string;
  domain: string;                // domain name
  available: boolean;            // available for registration
  can_register: boolean;         // user can register it
  current_registrar: string;     // current registrar name
  expires_at: string;            // expiration timestamp
  locked: boolean;               // transfer lock enabled
  registrant_contact: {
    // contact address fields
  };
  registry_statuses: string[];   // ICANN status codes
  supported_tld: boolean;        // TLD supported by cloudflare registrar
  transfer_in: {                 // transfer status
    accept_foa: string;
    enter_auth_code: string;
    unlock_domain: string;
  };
  updated_at: string;            // timestamp
  created_at: string;            // timestamp
}
```

**unique key**: `domain` (domain name is unique)
**primary key**: `domain`

---

### literal: WhoisRecord

**cloudflare api shape** [4]:
```typescript
{
  domain: string;
  extension: string;             // TLD
  punycode: string;              // IDN representation
  found: boolean;                // domain registered somewhere
  registrar: string;             // current registrar
  registrar_id: string;
  registrar_name: string;
  nameservers: string[];
  registrant: object;            // contact info
  created_date: string;
  updated_date: string;
  expiration_date: string;
  dnssec: boolean;
  status: string[];              // ICANN status codes
  whois_server: string;
}
```

**note**: whois records are read-only lookups, not managed entities

## domain operations

### zone operations

| operation | cloudflare method | path | declastruct pattern |
|-----------|-------------------|------|---------------------|
| getAll | GET | `/zones` | `daoZone.getAll()` |
| getOne | GET | `/zones/{zone_id}` | `daoZone.getOne({ by: { primary | unique } })` |
| set (create) | POST | `/zones` | `daoZone.set({ findsert })` |
| set (update) | PATCH | `/zones/{zone_id}` | `daoZone.set({ upsert })` |
| del | DELETE | `/zones/{zone_id}` | `daoZone.del({ by })` |

### dns record operations

| operation | cloudflare method | path | declastruct pattern |
|-----------|-------------------|------|---------------------|
| getAll | GET | `/zones/{zone_id}/dns_records` | `daoDnsRecord.getAll({ by: { zone } })` |
| getOne | GET | `/zones/{zone_id}/dns_records/{id}` | `daoDnsRecord.getOne({ by: { primary | unique } })` |
| set (create) | POST | `/zones/{zone_id}/dns_records` | `daoDnsRecord.set({ findsert })` |
| set (update) | PUT/PATCH | `/zones/{zone_id}/dns_records/{id}` | `daoDnsRecord.set({ upsert })` |
| del | DELETE | `/zones/{zone_id}/dns_records/{id}` | `daoDnsRecord.del({ by })` |

### registration operations

| operation | cloudflare method | path | declastruct pattern |
|-----------|-------------------|------|---------------------|
| getAll | GET | `/accounts/{account_id}/registrar/domains` | `daoRegistration.getAll()` |
| getOne | GET | `/accounts/{account_id}/registrar/domains/{name}` | `daoRegistration.getOne({ by: { unique } })` |
| set (update) | PUT | `/accounts/{account_id}/registrar/domains/{name}` | `daoRegistration.set({ upsert })` |

**note**: registration create not supported via api — requires HelpfulError guidance

### whois operations

| operation | cloudflare method | path | declastruct pattern |
|-----------|-------------------|------|---------------------|
| getOne | GET | `/accounts/{account_id}/intel/whois?domain=...` | `daoWhois.getOne({ domain })` |

## relationships

### dependency treestruct

```
Zone
├── DnsRecord[]           // zone contains many dns records
│   └── zone: RefByUnique<Zone>
└── Registration?         // zone may have associated registration
    └── zone: RefByUnique<Zone>
```

### operation order dependencies

1. **zone before record**: dns record creation requires zone to exist
2. **record before zone delete**: dns records must be deleted before zone
3. **zone before registration**: registration requires active cloudflare zone (for transfer)

### cross-entity references

| from entity | to entity | reference type | purpose |
|-------------|-----------|----------------|---------|
| DnsRecord | Zone | RefByUnique({ name }) | identifies parent zone |
| Registration | Zone | RefByUnique({ name }) | associates registration with zone |

## composition

### use case: zone + dns management

```
User declares:
  Zone { name: 'example.com', type: 'full' }
  DnsRecord { zone: { name: 'example.com' }, name: 'api', type: 'A', content: '...' }

Declastruct:
  1. plan: compare declared vs actual
  2. apply: create zone first (if absent), then create record
```

### use case: registration + transfer guidance

```
User declares:
  Registration { name: 'example.com', autoRenew: true, ... }

Declastruct:
  1. check if domain in cloudflare registrar (via daoRegistration.getOne)
  2. if absent, check whois (via daoWhois.getOne)
  3. if registered elsewhere → throw HelpfulError (transfer guidance)
  4. if unregistered → throw HelpfulError (purchase guidance)
  5. if in cloudflare → update settings via daoRegistration.set({ upsert })
```
