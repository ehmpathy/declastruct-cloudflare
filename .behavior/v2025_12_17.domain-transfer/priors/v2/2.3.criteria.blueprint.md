## subcomponent contracts

### domain objects

```
given('DeclaredCloudflareDomainZone contract')
  then('static unique = ["name"]')
  then('static primary = ["id"]')
  then('exposes: name, type, id?')
    sothat('zones are identified by domain name')

given('DeclaredCloudflareDomainDnsRecord contract')
  then('static unique = ["zone", "name", "type"]')
  then('static primary = ["id"]')
  then('exposes: zone (RefByUnique), name, type, content, ttl, proxied?, priority?')
    sothat('records are identified by zone + name + type composite')

given('DeclaredCloudflareDomainRegistration contract')
  then('static unique = ["name"]')
  then('static primary = ["name"]')
  then('exposes: name, zone (RefByUnique), autoRenew, locked, privacyProtection')
    sothat('registration settings are identified by domain name')
```

### dao operations

```
given('daoZone contract')
  then('exposes: getAll({ accountId }) => Zone[]')
  then('exposes: getOne({ by: { primary | unique | ref } }) => Zone | null')
  then('exposes: set({ findsert | upsert }) => Zone')
  then('exposes: del({ by }) => void')
    sothat('zones support full CRUD')

given('daoDnsRecord contract')
  then('exposes: getAll({ by: { zone } }) => DnsRecord[]')
  then('exposes: getOne({ by: { primary | unique | ref } }) => DnsRecord | null')
  then('exposes: set({ findsert | upsert }) => DnsRecord')
  then('exposes: del({ by }) => void')
    sothat('dns records support full CRUD')

given('daoRegistration contract')
  then('exposes: getAll({ accountId }) => Registration[]')
  then('exposes: getOne({ by: { primary | unique | ref } }) => Registration | null')
  then('exposes: set({ findsert | upsert }) => Registration')
    sothat('registrations support list and update')
  then('throws HelpfulError for transfer guidance when domain registered elsewhere')
  then('throws HelpfulError for purchase guidance when domain unregistered')
    sothat('guidance is provided when automation is blocked')
```

given('getDeclastructCloudflareProvider contract')

## composition boundaries

```
given('declastruct integration')
  then('provider exposes daos')
  then('daos use cloudflare sdk internally')
  then('domain objects are serialized/deserialized at dao boundary')
    sothat('declastruct controls plan and order of operations')

given('zone + dns record dependency')
  then('dns records reference zones via RefByUnique')
  then('declastruct orders operations: create zone before record, delete record before zone')
    sothat('dependencies are handled automatically')

given('registration + whois integration')
  then('registration dao calls whois api when domain not found in cloudflare')
  then('whois response determines transfer vs purchase guidance')
    sothat('correct guidance is shown based on domain status')
```
