# distill.domain: declastruct-cloudflare domain model

## usecases overview

| usecase | domain objects | operations |
|---------|----------------|------------|
| zone management | DeclaredCloudflareDomainZone | getAll, getOne, set, del |

## domain.objects

### entity.1: DeclaredCloudflareDomainZone

**purpose**: dns container, gateway to registrar

**shape**:
```typescript
interface DeclaredCloudflareDomainZone {
  id?: string;                    // @metadata — assigned by cloudflare
  name: string;                   // @unique — domain name
  type: 'full' | 'partial' | 'secondary';
  status?: ZoneStatus;            // @readonly — assigned by cloudflare
  nameServers?: string[];         // @readonly — assigned by cloudflare
  createdOn?: Date;               // @readonly
}

class DeclaredCloudflareDomainZone extends DomainEntity<DeclaredCloudflareDomainZone> {
  public static primary = ['id'] as const;
  public static unique = ['name'] as const;
  public static metadata = ['id'] as const;
  public static readonly = ['status', 'nameServers', 'createdOn'] as const;
}
```

**identity**:
- primary key: `id` (cloudflare-assigned)
- unique key: `name` (domain name)

### entity.2: DeclaredCloudflareDomainDnsRecord

**purpose**: individual dns entry within a zone

**shape**:
```typescript
interface DeclaredCloudflareDomainDnsRecord {
  id?: string;                                        // @metadata
  zone: RefByUnique<typeof DeclaredCloudflareDomainZone>;  // { name: string }
  name: string;                                       // record name (fqdn)
  type: DnsRecordType;                                // A, AAAA, CNAME, MX, TXT, etc
  content: string;                                    // record value
  ttl: number;                                        // 1 = automatic, 60-86400
  proxied?: boolean;                                  // cloudflare proxy
  priority?: number;                                  // for MX, SRV
  comment?: string | null;                            // optional note
  createdOn?: Date;                                   // @readonly
  modifiedOn?: Date;                                  // @readonly
}

class DeclaredCloudflareDomainDnsRecord extends DomainEntity<DeclaredCloudflareDomainDnsRecord> {
  public static primary = ['id'] as const;
  public static unique = ['zone', 'name', 'type'] as const;  // composite
  public static metadata = ['id'] as const;
  public static readonly = ['createdOn', 'modifiedOn'] as const;
}
```

**identity**:
- primary key: `id` (cloudflare-assigned)
- unique key: `(zone, name, type)` (composite)

### entity.3: DeclaredCloudflareDomainRegistration

**purpose**: registrar-level domain ownership and settings

**shape**:
```typescript
interface DeclaredCloudflareDomainRegistration {
  name: string;                                       // @unique @primary

  autoRenew: boolean;
  locked: boolean;
  privacyProtection: boolean;
  expiresAt?: Date;                                   // @readonly
  registrantContact?: RegistrantContact | null;       // @readonly

class DeclaredCloudflareDomainRegistration extends DomainEntity<DeclaredCloudflareDomainRegistration> {
  public static primary = ['name'] as const;
  public static unique = ['name'] as const;
  public static readonly = ['expiresAt', 'registrantContact', 'status'] as const;
}
```

**identity**:
- primary key: `name` (domain name)
- unique key: `name` (domain name)

## domain.operations

### operations.zone

| operation | signature | purpose |
|-----------|-----------|---------|
| getAllDomainZones | `(context) => Zone[]` | list all zones in account |
| getOneDomainZone | `({ by: { primary \| unique \| ref } }, context) => Zone \| null` | get single zone |
| setDomainZone | `({ findsert \| upsert }, context) => Zone` | create/update zone |
| delDomainZone | `({ by }, context) => void` | delete zone |

### operations.dnsRecord

| operation | signature | purpose |
|-----------|-----------|---------|
| getAllDomainDnsRecords | `({ by: { zone } }, context) => DnsRecord[]` | list records for zone |
| getOneDomainDnsRecord | `({ by: { primary \| unique \| ref } }, context) => DnsRecord \| null` | get single record |
| setDomainDnsRecord | `({ findsert \| upsert }, context) => DnsRecord` | create/update record |
| delDomainDnsRecord | `({ by }, context) => void` | delete record |

### operations.registration

| operation | signature | purpose |
|-----------|-----------|---------|
| getAllDomainRegistrations | `(context) => Registration[]` | list all registrations |
| getOneDomainRegistration | `({ by: { unique \| ref } }, context) => Registration \| null` | get single registration |
| setDomainRegistration | `({ findsert \| upsert }, context) => Registration` | update settings OR throw guidance |

**special behavior for setDomainRegistration**:

```
if domain in cloudflare registrar:
  → apply settings, return registration

if domain NOT in cloudflare registrar:
  → call WHOIS API to determine status
  → if registered elsewhere:
      throw BadRequestError with transfer guidance
  → if unregistered:
      throw BadRequestError with purchase guidance
```

### operations.whois

| operation | signature | purpose |
|-----------|-----------|---------|
| getOneWhoisRecord | `({ domain }, context) => WhoisRecord \| null` | lookup domain status |

## relationships

### treestruct

```
DeclaredCloudflare
└── Domain
    ├── Zone              // dns container
    │   └── DnsRecord[]   // child records
    └── Registration      // ownership settings
```

### dependencies

```
Zone
├── DnsRecord depends on Zone (must create zone before records)
└── Registration depends on Zone (zone must be active for transfer)
```

### operation order

| scenario | required order |
|----------|----------------|
| create zone + records | zone first, then records |
| delete zone + records | records first, then zone |
| transfer domain | zone must be active, then registration |

## composition

### usecase: zone + dns management

```typescript
const getResources = async () => {
  const zone = new DeclaredCloudflareDomainZone({
    name: 'example.com',
    type: 'full',
  });

  const apiRecord = new DeclaredCloudflareDomainDnsRecord({
    zone: refByUnique(zone),
    name: 'api.example.com',
    type: 'A',
    content: '192.168.1.100',
    ttl: 300,
    proxied: true,
  });

  return [zone, apiRecord];
};
```

### usecase: registration with guidance

```typescript
const getResources = async () => {
  const zone = new DeclaredCloudflareDomainZone({
    name: 'example.com',
    type: 'full',
  });

  const registration = new DeclaredCloudflareDomainRegistration({
    name: 'example.com',
    zone: refByUnique(zone),
    autoRenew: true,
    locked: true,
    privacyProtection: true,
  });

  return [zone, registration];
};

// on apply:
// - if domain not in cloudflare registrar → HelpfulError with guidance
// - if domain in cloudflare registrar → settings applied
```
