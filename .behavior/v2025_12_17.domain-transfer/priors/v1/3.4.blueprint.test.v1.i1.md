# blueprint: test coverage for domain-transfer wish

## .what

blueprint for proving fulfillment of the domain-transfer wish via comprehensive test coverage at each grain.

## .context

wish = domain transfer & management in cloudflare via declastruct declarative control

## .test preconditions

### demo account constraints

the demo cloudflare account has limitations:
- **no zones**: cannot test zone/record operations that require existing zones
- **no domains**: cannot test registration operations

### test resource provisioning

for full coverage, tests need:
1. **test zone**: a zone owned by the test account (e.g., `test-declastruct.dev`)
2. **test domain**: a domain registered in cloudflare for registration tests

### fail-fast on missing preconditions

```typescript
// pattern for tests requiring resources
const scene = useBeforeAll(async () => {
  const zones = await getAllDomainZones(context);
  if (!zones[0])
    throw new Error(
      'test precondition not met: no zones exist in account. ' +
      'provision a test zone or use an account with existing zones.',
    );
  return { zone: zones[0] };
});
```
